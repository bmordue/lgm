# Implementation Plan: Implement Damage Calculation Formulas

**Issue ID**: lgm-sqo
**Priority**: P2
**Type**: Task
**Depends On**: 
- lgm-ahi (Define damage calculation factors)
- lgm-22y (Define weapon types) - ✅ CLOSED

## 1. Summary

Implement the damage calculation formulas that determine how much damage attacks deal, incorporating all defined factors including weapon stats, distance modifiers, terrain effects, cover bonuses, armor reduction, and random variance.

## 2. Problem

The current damage system in `api/service/Rules.ts` uses a simplified approach:

```typescript
// Current implementation
allActorsInWorld[targetIndex].health -= allActorsInWorld[attackerIndex].weapon.damage;
```

This lacks:
- **Distance Modifiers**: Damage doesn't falloff at range
- **Terrain Effects**: No high ground or difficult terrain modifiers
- **Cover System**: Blocked terrain doesn't reduce damage
- **Armor**: No damage reduction from armor stats
- **Random Variance**: Damage is always exactly the weapon value
- **Penetration**: No armor piercing mechanic
- **Damage Breakdown**: No explanation of damage calculation

### Current State
From `api/service/Rules.ts`:
```typescript
async function applyFiringRules(...) {
    // ... validation ...
    
    // Simple flat damage
    allActorsInWorld[targetIndex].health -= allActorsInWorld[attackerIndex].weapon.damage;
    
    if (allActorsInWorld[targetIndex].health <= 0) {
        allActorsInWorld[targetIndex].state = ActorState.DEAD;
    }
}
```

### Impact
Without proper damage calculation:
- Combat is predictable and boring
- No tactical positioning value
- Armor and cover meaningless
- No excitement from variance
- Weapon balance impossible

## 3. Proposed Solution

Implement comprehensive damage calculation using the formula defined in lgm-ahi:

```
finalDamage = baseDamage × distanceMod × terrainMod × coverMod × armorMod × randomMod

where:
  baseDamage = weapon.baseDamage
  distanceMod = distance falloff (0.0 - 1.0)
  terrainMod = 1.0 + (attackerTerrainBonus / 100)
  coverMod = 1.0 - (defenderCoverBonus / 100)
  armorMod = 1.0 - (effectiveArmor / 100)
  randomMod = 0.85 + (Math.random() * 0.3)  // 85% - 115%
```

### Key Components

1. **CombatMath Module**: Pure functions for damage calculation
2. **Damage Breakdown**: Human-readable explanation of calculation
3. **Configuration-Driven**: All modifiers from config files
4. **Testable**: Deterministic testing with seeded random

import { Actor, Weapon, Terrain, GridPosition } from './Models';
import { Hex } from '../Hex';
import { getCombatConfig, CombatConfig } from '../config/CombatConfig';
import { getTerrainProperties, TerrainProperties } from '../config/TerrainConfig';

export interface DamageCalculation {
    // Inputs
    baseDamage: number;
    distance: number;
    
    // Modifiers
    distanceModifier: number;
    terrainModifier: number;
    coverModifier: number;
    armorModifier: number;
    randomModifier: number;
    
    // Result
    finalDamage: number;
    
    // Debug info
    breakdown: string;
    isCritical: boolean;
    criticalMultiplier?: number;
}

export interface DamageOptions {
    attackerPos?: GridPosition;
    defenderPos?: GridPosition;
    hasLineOfSight?: boolean;
    isOverwatch?: boolean;
    isMelee?: boolean;
    forceCritical?: boolean;
}

/**
 * Calculate damage dealt from attacker to defender
 */
export function calculateDamage(
    attacker: Actor,
    defender: Actor,
    distance: number,
    attackerTerrain: Terrain,
    defenderTerrain: Terrain,
    hasLineOfSight: boolean = true,
    options: DamageOptions = {}
): DamageCalculation {
    const config = getCombatConfig();
    
    // Check for no weapon
    if (!attacker.weapon) {
        return createZeroDamageResult('Attacker has no weapon');
    }

    const weapon = attacker.weapon;

    // Check line of sight
    if (!hasLineOfSight) {
        return createZeroDamageResult('No line of sight');
    }

    // Check range
    if (distance < weapon.minRange || distance > weapon.maxRange) {
        return createZeroDamageResult(`Target out of range (distance: ${distance}, valid: ${weapon.minRange}-${weapon.maxRange})`);
    }

    // Base damage from weapon
    const baseDamage = weapon.baseDamage || weapon.damage || 10;

    // Distance modifier
    const distanceMod = calculateDistanceModifier(weapon, distance, config);

    // Terrain modifiers
    const attackerTerrainProps = getTerrainProperties(attackerTerrain);
    const defenderTerrainProps = getTerrainProperties(defenderTerrain);
    
    const terrainMod = calculateTerrainModifier(attackerTerrainProps, defenderTerrainProps, config);
    const coverMod = calculateCoverModifier(defenderTerrainProps, defender, config);

    // Armor reduction
    const armorMod = calculateArmorModifier(weapon, defender, config);

    // Random variance
    let randomMod = calculateRandomModifier(config);
    
    // Critical hit check
    const isCritical = options.forceCritical || checkCriticalHit(config);
    const criticalMod = isCritical ? config.criticalHitMultiplier : 1.0;

    // Overwatch penalty
    if (options.isOverwatch) {
        randomMod *= config.overwatchAccuracy;  // Typically 0.7
    }

    // Aim bonus
    if (attacker.isAiming && attacker.aimBonus) {
        randomMod *= (1.0 + attacker.aimBonus / 100);
    }

    // Calculate final damage
    const finalDamage = calculateFinalDamage(
        baseDamage,
        distanceMod,
        terrainMod,
        coverMod,
        armorMod,
        randomMod,
        criticalMod,
        config
    );

    return {
        baseDamage,
        distance,
        distanceModifier: distanceMod,
        terrainModifier: terrainMod,
        coverModifier: coverMod,
        armorModifier: armorMod,
        randomModifier: randomMod,
        finalDamage,
        isCritical,
        criticalMultiplier: isCritical ? criticalMod : undefined,
        breakdown: createDamageBreakdown(
            baseDamage,
            distanceMod,
            terrainMod,
            coverMod,
            armorMod,
            randomMod,
            criticalMod,
            finalDamage
        )
    };
}

/**
 * Calculate distance-based damage falloff
 */
export function calculateDistanceModifier(
    weapon: Weapon,
    distance: number,
    config: CombatConfig
): number {
    // Out of range
    if (distance < weapon.minRange || distance > weapon.maxRange) {
        return 0;
    }

    // Determine optimal range
    const optimalRange = weapon.optimalRange || 
        Math.floor((weapon.minRange + weapon.maxRange) / 2);

    // At optimal range = full damage
    if (distance === optimalRange) {
        return 1.0;
    }

    // Inside optimal range (close range penalty for some weapons)
    if (distance < optimalRange) {
        if (weapon.minRange === optimalRange) {
            return 1.0;  // No close range penalty
        }
        
        const rangeSpan = optimalRange - weapon.minRange;
        const distanceFromMin = distance - weapon.minRange;
        // Interpolate between 0.7 and 1.0
        return config.minRangePenalty + 
            ((1.0 - config.minRangePenalty) * (distanceFromMin / rangeSpan));
    }

    // Beyond optimal range (falloff)
    const rangeSpan = weapon.maxRange - optimalRange;
    const distanceFromOptimal = distance - optimalRange;
    // Interpolate between 1.0 and maxRangePenalty
    return 1.0 - ((1.0 - config.maxRangePenalty) * (distanceFromOptimal / rangeSpan));
}

/**
 * Calculate terrain-based modifier
 */
export function calculateTerrainModifier(
    attackerTerrain: TerrainProperties,
    defenderTerrain: TerrainProperties,
    config: CombatConfig
): number {
    // High ground bonus
    let modifier = 1.0;
    
    if (attackerTerrain.accuracyModifier) {
        modifier += attackerTerrain.accuracyModifier / 100;
    }
    
    if (defenderTerrain.accuracyModifier) {
        modifier -= defenderTerrain.accuracyModifier / 100;  // Defender bonus = attacker penalty
    }

    // Clamp to reasonable range
    return Math.max(config.minAccuracyModifier, Math.min(config.maxAccuracyModifier, modifier));
}

/**
 * Calculate cover-based damage reduction
 */
export function calculateCoverModifier(
    defenderTerrain: TerrainProperties,
    defender: Actor,
    config: CombatConfig
): number {
    let coverBonus = defenderTerrain.coverBonus || 0;
    
    // Actor under cover action
    if (defender.isUnderCover && defender.coverBonus) {
        coverBonus = Math.max(coverBonus, defender.coverBonus);
    }

    // Apply cover effectiveness config
    const effectiveCover = coverBonus * config.coverEffectiveness;
    
    return 1.0 - (effectiveCover / 100);
}

/**
 * Calculate armor-based damage reduction
 */
export function calculateArmorModifier(
    weapon: Weapon,
    defender: Actor,
    config: CombatConfig
): number {
    const armorValue = defender.armor || 0;
    const penetration = weapon.penetration || 0;
    
    // Effective armor after penetration
    const effectiveArmor = Math.max(0, armorValue - penetration);
    
    // Cap armor reduction
    const cappedArmor = Math.min(effectiveArmor, config.maxArmorReduction);
    
    return 1.0 - (cappedArmor / 100);
}

/**
 * Calculate random variance modifier
 */
export function calculateRandomModifier(config: CombatConfig): number {
    return config.damageVariance.min + 
        (Math.random() * (config.damageVariance.max - config.damageVariance.min));
}

/**
 * Check if hit is critical
 */
export function checkCriticalHit(config: CombatConfig): boolean {
    return Math.random() < config.criticalHitChance;
}

/**
 * Calculate final damage value
 */
function calculateFinalDamage(
    baseDamage: number,
    distanceMod: number,
    terrainMod: number,
    coverMod: number,
    armorMod: number,
    randomMod: number,
    criticalMod: number,
    config: CombatConfig
): number {
    const rawDamage = baseDamage * distanceMod * terrainMod * coverMod * armorMod * randomMod * criticalMod;
    
    // Round and apply minimum
    const finalDamage = Math.max(config.minDamage, Math.floor(rawDamage));
    
    return finalDamage;
}

/**
 * Create human-readable damage breakdown
 */
export function createDamageBreakdown(
    base: number,
    distance: number,
    terrain: number,
    cover: number,
    armor: number,
    random: number,
    critical: number,
    final: number
): string {
    const parts: string[] = [];
    
    parts.push(`Base: ${base}`);
    
    if (distance !== 1.0) {
        parts.push(`× Range: ${(distance * 100).toFixed(0)}%`);
    }
    
    if (terrain !== 1.0) {
        parts.push(`× Terrain: ${(terrain * 100).toFixed(0)}%`);
    }
    
    if (cover !== 1.0) {
        parts.push(`× Cover: ${(cover * 100).toFixed(0)}%`);
    }
    
    if (armor !== 1.0) {
        parts.push(`× Armor: ${(armor * 100).toFixed(0)}%`);
    }
    
    if (random !== 1.0) {
        parts.push(`× Variance: ${(random * 100).toFixed(0)}%`);
    }
    
    if (critical > 1.0) {
        parts.push(`× CRIT: ${(critical * 100).toFixed(0)}%`);
    }
    
    parts.push(`= ${final}`);
    
    return parts.join(' ');
}

function createZeroDamageResult(reason: string): DamageCalculation {
    return {
        baseDamage: 0,
        distance: 0,
        distanceModifier: 0,
        terrainModifier: 0,
        coverModifier: 0,
        armorModifier: 0,
        randomModifier: 0,
        finalDamage: 0,
        isCritical: false,
        breakdown: reason
    };
}

/**
 * Calculate expected damage (for AI, UI previews)
 * Uses average random modifier instead of rolling
 */
export function calculateExpectedDamage(
    attacker: Actor,
    defender: Actor,
    distance: number,
    attackerTerrain: Terrain,
    defenderTerrain: Terrain
): number {
    const config = getCombatConfig();
    const avgRandom = (config.damageVariance.min + config.damageVariance.max) / 2;
    
    const result = calculateDamage(
        attacker,
        defender,
        distance,
        attackerTerrain,
        defenderTerrain,
        true,
        { }
    );
    
    // Recalculate with average random
    return Math.floor(result.finalDamage / result.randomModifier * avgRandom);
}

import { calculateDamage } from './CombatMath';
import { gridPositionToHex } from './Visibility';

async function applyFiringRules(
    actorOrders: ActorOrders,
    game: Game,
    world: World,
    allActorsInWorld: Actor[],
    timestep: number
): Promise<Actor> {
    const attackerIndex = allActorsInWorld.findIndex(a => a.id === actorOrders.actorId);
    if (attackerIndex === -1) {
        throw new Error(`Attacker actor with ID ${actorOrders.actorId} not found`);
    }

    if (actorOrders.orderType !== OrderType.ATTACK) {
        return allActorsInWorld[attackerIndex];
    }

    if (typeof actorOrders.targetId === 'undefined') {
        logger.debug(`Actor ${allActorsInWorld[attackerIndex].id} has ATTACK order but no targetId.`);
        return allActorsInWorld[attackerIndex];
    }

    const attacker = allActorsInWorld[attackerIndex];
    const targetIndex = allActorsInWorld.findIndex(a => a.id === actorOrders.targetId);

    if (targetIndex === -1) {
        logger.warn(`ATTACK order: Target actor with ID ${actorOrders.targetId} not found`);
        return attacker;
    }

    const target = allActorsInWorld[targetIndex];

    // Validate target
    if (target.state === ActorState.DEAD) {
        logger.info(`ATTACK order: Target actor ${target.id} is already dead.`);
        return attacker;
    }

    if (attacker.id === target.id) {
        logger.info(`ATTACK order: Actor cannot target itself.`);
        return attacker;
    }

    if (!attacker.weapon) {
        logger.error(`Actor ${attacker.id} has no weapon, cannot attack.`);
        return attacker;
    }

    // Calculate positions and distance
    const attackerHex = gridPositionToHex(attacker.pos);
    const targetHex = gridPositionToHex(target.pos);
    const distance = attackerHex.distance(targetHex);

    // Check weapon range
    if (distance < attacker.weapon.minRange || distance > attacker.weapon.maxRange) {
        logger.info(`ATTACK order: Target out of range (distance: ${distance})`);
        return attacker;
    }

    // Check line of sight
    const losClear = hasLineOfSight(attackerHex, targetHex, world.terrain, allActorsInWorld);
    if (!losClear) {
        logger.info(`ATTACK order: Line of sight is blocked.`);
        return attacker;
    }

    // Get terrain at positions
    const attackerTerrain = world.terrain[attacker.pos.x]?.[attacker.pos.y] || Terrain.EMPTY;
    const targetTerrain = world.terrain[target.pos.x]?.[target.pos.y] || Terrain.EMPTY;

    // Calculate damage using new system
    const damageCalc = calculateDamage(
        attacker,
        target,
        distance,
        attackerTerrain,
        targetTerrain,
        losClear
    );

    // Apply damage
    target.health = Math.max(0, (target.health || 100) - damageCalc.finalDamage);

    logger.info(util.format(
        'Actor %d attacked Actor %d: %d damage (HP: %d -> %d) [%s]',
        attacker.id,
        target.id,
        damageCalc.finalDamage,
        (target.health || 0) + damageCalc.finalDamage,
        target.health,
        damageCalc.breakdown
    ));

    // Check for kill
    if (target.health <= 0) {
        target.state = ActorState.DEAD;
        logger.info(`Actor ${target.id} has been defeated.`);
    }

    // Consume ammo
    if (attacker.weapon.ammo) {
        attacker.currentAmmo = (attacker.currentAmmo || attacker.weapon.ammo) - 1;
        if (attacker.currentAmmo <= 0) {
            logger.info(`Actor ${attacker.id} is out of ammo!`);
        }
    }

    return attacker;
}

            const noArmorDamage = CombatMath.calculateDamage(
                attacker,
                noArmor,
                2,
                Terrain.EMPTY,
                Terrain.EMPTY,
                true
            );
            
            const armorDamage = CombatMath.calculateDamage(
                attacker,
                heavyArmor,
                2,
                Terrain.EMPTY,
                Terrain.EMPTY,
                true
            );
            
            expect(armorDamage.finalDamage).to.be.lt(noArmorDamage.finalDamage);
        });

        it('should account for armor penetration', () => {
            const attacker = createTestActor(1, { x: 0, y: 0 }, 1, 'RIFLE');
            attacker.weapon!.penetration = 30;
            
            const defender = createTestActor(2, { x: 5, y: 0 }, 2, 'PISTOL', {
                armor: 40
            });
            
            const damage = CombatMath.calculateDamage(
                attacker,
                defender,
                5,
                Terrain.EMPTY,
                Terrain.EMPTY,
                true
            );
            
            // Effective armor should be 40 - 30 = 10
            expect(damage.armorModifier).to.be.closeTo(0.9, 0.15);
        });
    });

    describe('calculateDamage - Random Variance', () => {
        it('should have damage variance between 85% and 115%', () => {
            const attacker = createTestActor(1, { x: 0, y: 0 }, 1, 'RIFLE');
            const defender = createTestActor(2, { x: 5, y: 0 }, 2);
            
            const damages: number[] = [];
            
            for (let i = 0; i < 50; i++) {
                const damage = CombatMath.calculateDamage(
                    attacker,
                    defender,
                    5,
                    Terrain.EMPTY,
                    Terrain.EMPTY,
                    true
                );
                damages.push(damage.finalDamage);
            }
            
            const minDamage = Math.min(...damages);
            const maxDamage = Math.max(...damages);
            
            // Should see variance
            expect(maxDamage).to.be.gte(minDamage);
            
            // Max should be roughly 1.35x min (115/85)
            const ratio = maxDamage / minDamage;
            expect(ratio).to.be.lte(1.5);  // Some buffer for small sample
        });
    });

    describe('calculateDistanceModifier', () => {
        it('should return 1.0 at optimal range', () => {
            const weapon = { ...WEAPON_TYPES.RIFLE };
            const config = CombatMath.getCombatConfig();
            
            const mod = CombatMath.calculateDistanceModifier(
                weapon,
                weapon.optimalRange || 5,
                config
            );
            
            expect(mod).to.equal(1.0);
        });

        it('should falloff beyond optimal range', () => {
            const weapon = { ...WEAPON_TYPES.RIFLE };
            const config = CombatMath.getCombatConfig();
            
            const optimalMod = CombatMath.calculateDistanceModifier(
                weapon,
                weapon.optimalRange || 5,
                config
            );
            
            const farMod = CombatMath.calculateDistanceModifier(
                weapon,
                weapon.maxRange,
                config
            );
            
            expect(farMod).to.be.lt(optimalMod);
        });

        it('should return 0 outside effective range', () => {
            const weapon = { ...WEAPON_TYPES.PISTOL };
            const config = CombatMath.getCombatConfig();
            
            const mod = CombatMath.calculateDistanceModifier(
                weapon,
                weapon.maxRange + 1,
                config
            );
            
            expect(mod).to.equal(0);
        });
    });

    describe('createDamageBreakdown', () => {
        it('should create readable damage breakdown', () => {
            const breakdown = CombatMath.createDamageBreakdown(

                1.0, // terrain
                0.7, // cover
                0.8, // armor
                0.95,// random
                1.0, // critical
                9    // final
            );
            
            expect(breakdown).to.include('Base: 20');
            expect(breakdown).to.include('Cover: 70%');
            expect(breakdown).to.include('Armor: 80%');
            expect(breakdown).to.include('= 9');
        });
    });

# Damage Calculation System

## Overview
The LGM damage system calculates combat damage using multiple factors...

## Formula
```
finalDamage = baseDamage × distanceMod × terrainMod × coverMod × armorMod × randomMod × criticalMod
```

## Factors

### Base Damage
Weapon's base damage value from WeaponsConfig.

### Distance Modifier
- Optimal range = 100% damage
- Min to optimal = 70% to 100%
- Optimal to max = 100% to 50%

### Terrain Modifier
- High ground = +20% accuracy
- Difficult terrain = -10% to -20%

### Cover Modifier
- Open ground = 0%
- Light cover = 20-30%
- Heavy cover = 40-50%

### Armor Modifier
- Each armor point = 1% reduction
- Penetration reduces effective armor
- Max 50% reduction

### Random Variance
- 85% to 115% of calculated damage
- Adds unpredictability

### Critical Hits
- 5% chance
- 2.0x multiplier

### Example 1: Standard Attack

= 12 damage (±15% variance)

Effective armor = 40 - 20 = 20
