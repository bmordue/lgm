# Implementation Plan: Add WebSocket Support for Real-time Updates

**Issue ID**: lgm-j24
**Priority**: P2
**Type**: Feature
**Dependencies**: 
- lgm-8dy (Implement database integration)

## 1. Summary

Implement real-time communication using WebSockets (Socket.IO) to provide instant game state updates to clients, replacing inefficient HTTP polling with event-driven push notifications.

## 2. Problem

Currently, the frontend has no way to know when game state changes unless it actively polls the server:

- **Latency**: Players wait several seconds to see turn results
- **Inefficiency**: Constant polling creates unnecessary network traffic
- **Poor UX**: Game feels sluggish and unresponsive
- **Server Load**: Many requests return no new data

### Current State
From examining the codebase:
- Frontend uses Pinia stores (`client/src/stores/Games.store.ts`, `User.store.ts`)
- Backend has controllers (`api/controllers/GameController.ts`, `UsersController.ts`)
- No WebSocket infrastructure exists
- Frontend likely polls for updates (or requires manual refresh)

### Impact
Without real-time updates:
- Multiplayer experience is broken
- Players don't see opponents' moves
- Turn results not delivered promptly
- Game state desynchronization

## 3. Proposed Solution

Integrate Socket.IO for bidirectional real-time communication:

### Events
```
Server → Client Events:
- game:update         (turn processed, game state changed)
- game:player_joined  (new player joined)
- game:player_left    (player left game)
- game:started        (game began)
- game:ended          (game completed)
- notification        (generic notification)
- error               (error message)

Client → Server Events:
- join_game           (join game room)
- leave_game          (leave game room)
- ping                (keepalive)
```

### Architecture
```
┌─────────────┐     WebSocket      ┌─────────────┐
│   Client    │ ◄────────────────► │   Server    │
│  (Vue/Pinia)│     Socket.IO      │ (Express +  │
└─────────────┘                    │  Socket.IO) │
                                   └─────────────┘
                                          │
                                          ▼
                                   ┌─────────────┐
                                   │   Game      │
                                   │  Services   │
                                   └─────────────┘
```

#### 2.1 Create WebSocket Service
Create `api/service/WebSocketService.ts`:

```typescript
import { Server as SocketIOServer, Socket } from 'socket.io';
import { Server as HTTPServer } from 'http';
import logger = require('../utils/Logger');
import util = require('util');

export interface JoinGamePayload {
    gameId: number;
    playerId: number;
}

export interface GameUpdatePayload {
    gameId: number;
    turn: number;
    world?: any;
    combatEvents?: any[];
}

export interface PlayerJoinedPayload {
    gameId: number;
    playerId: number;
    username: string;
}

class WebSocketService {
    private io: SocketIOServer | null = null;
    private connectedClients: Map<string, Socket> = new Map();

    /**
     * Initialize Socket.IO server
     */
    public init(httpServer: HTTPServer): void {
        this.io = new SocketIOServer(httpServer, {
            cors: {
                origin: process.env.WS_CORS_ORIGIN || 'http://localhost:5173',
                methods: ['GET', 'POST']
            },
            pingTimeout: 60000,
            pingInterval: 25000
        });

        this.io.on('connection', (socket: Socket) => {
            this.handleConnection(socket);
        });

        logger.info('WebSocket (Socket.IO) server initialized');
    }

    /**
     * Handle new socket connection
     */
    private handleConnection(socket: Socket): void {
        logger.info(util.format('Client connected: %s', socket.id));

        this.connectedClients.set(socket.id, socket);

        // Join game room
        socket.on('join_game', (payload: JoinGamePayload) => {
            this.handleJoinGame(socket, payload);
        });

        // Leave game room
        socket.on('leave_game', (payload: { gameId: number }) => {
            this.handleLeaveGame(socket, payload.gameId);
        });

        // Ping/pong for keepalive
        socket.on('ping', () => {
            socket.emit('pong');
        });

        // Handle disconnect
        socket.on('disconnect', () => {
            this.handleDisconnect(socket);
        });

        // Handle errors
        socket.on('error', (error: Error) => {
            logger.error(util.format('Socket error: %s', error.message));
        });
    }

    /**
     * Handle join game room
     */
    private handleJoinGame(socket: Socket, payload: JoinGamePayload): void {
        const roomName = `game-${payload.gameId}`;
        socket.join(roomName);
        logger.info(util.format(
            'Client %s joined room %s (player %s)',
            socket.id, roomName, payload.playerId
        ));
    }

    /**
     * Handle leave game room
     */
    private handleLeaveGame(socket: Socket, gameId: number): void {
        const roomName = `game-${gameId}`;
        socket.leave(roomName);
        logger.info(util.format('Client %s left room %s', socket.id, roomName));
    }

    /**
     * Handle client disconnect
     */
    private handleDisconnect(socket: Socket): void {
        logger.info(util.format('Client disconnected: %s', socket.id));
        this.connectedClients.delete(socket.id);
    }

    /**
     * Emit event to all clients in game room
     */
    public emitToGame(gameId: number, event: string, data: any): void {
        if (!this.io) {
            logger.warn('Cannot emit to game: WebSocket server not initialized');
            return;
        }

        const roomName = `game-${gameId}`;
        this.io.to(roomName).emit(event, data);
        logger.debug(util.format('Emitted %s to room %s', event, roomName));
    }

    /**
     * Emit event to specific client
     */
    public emitToClient(socketId: string, event: string, data: any): void {
        const socket = this.connectedClients.get(socketId);
        if (socket) {
            socket.emit(event, data);
        }
    }

    /**
     * Emit game update to all players
     */
    public emitGameUpdate(payload: GameUpdatePayload): void {
        this.emitToGame(payload.gameId, 'game:update', payload);
    }

    /**
     * Emit player joined notification
     */
    public emitPlayerJoined(payload: PlayerJoinedPayload): void {
        this.emitToGame(payload.gameId, 'game:player_joined', payload);
    }

    /**
     * Emit notification to game
     */
    public emitNotification(gameId: number, message: string, type: string = 'info'): void {
        this.emitToGame(gameId, 'notification', { message, type });
    }

    /**
     * Get server instance (for testing)
     */
    public getServer(): SocketIOServer | null {
        return this.io;
    }
}

// Singleton instance
const webSocketService = new WebSocketService();

export function getWebSocketService(): WebSocketService {
    return webSocketService;
}

export default webSocketService;
```

#### 4.1 Create WebSocket Store
Create `client/src/stores/WebSocket.store.ts`:

```typescript
import { defineStore } from 'pinia';
import { io, Socket } from 'socket.io-client';

interface GameState {
    gameId: number;
    turn: number;
    world?: any;
}

interface Notification {
    message: string;
    type: 'info' | 'success' | 'warning' | 'error';
}

export const useWebSocketStore = defineStore('websocket', {
    state: () => ({
        socket: null as Socket | null,
        connected: false,
        currentGame: null as number | null,
        notifications: [] as Notification[]
    }),

    actions: {
        /**
         * Initialize WebSocket connection
         */
        connect() {
            const wsUrl = import.meta.env.VITE_WS_URL || 'http://localhost:3000';
            
            this.socket = io(wsUrl, {
                transports: ['websocket', 'polling'],
                reconnection: true,
                reconnectionDelay: 1000,
                reconnectionAttempts: 5
            });

            this.socket.on('connect', () => {
                console.log('WebSocket connected');
                this.connected = true;
            });

            this.socket.on('disconnect', () => {
                console.log('WebSocket disconnected');
                this.connected = false;
            });

            this.socket.on('game:update', (payload: GameState) => {
                this.handleGameUpdate(payload);
            });

            this.socket.on('game:player_joined', (payload: any) => {
                this.handlePlayerJoined(payload);
            });

            this.socket.on('notification', (notification: Notification) => {
                this.addNotification(notification);
            });

            this.socket.on('error', (error: any) => {
                console.error('WebSocket error:', error);
            });
        },

        /**
         * Join game room
         */
        joinGame(gameId: number, playerId: number) {
            if (this.socket) {
                this.socket.emit('join_game', { gameId, playerId });
                this.currentGame = gameId;
            }
        },

        /**
         * Leave game room
         */
        leaveGame() {
            if (this.socket && this.currentGame) {
                this.socket.emit('leave_game', { gameId: this.currentGame });
                this.currentGame = null;
            }
        },

        /**
         * Handle game update
         */
        handleGameUpdate(payload: GameState) {
            // Dispatch to games store
            const gamesStore = useGamesStore();
            gamesStore.updateGameState(payload);
            
            this.addNotification({
                message: `Turn ${payload.turn} processed!`,
                type: 'info'
            });
        },

        /**
         * Handle player joined
         */
        handlePlayerJoined(payload: any) {
            const gamesStore = useGamesStore();
            gamesStore.addPlayerToGame(payload.gameId, payload);
            
            this.addNotification({
                message: `${payload.username} joined the game`,
                type: 'success'
            });
        },

        /**
         * Add notification
         */
        addNotification(notification: Notification) {
            this.notifications.push(notification);
            // Auto-remove after 5 seconds
            setTimeout(() => {
                this.removeNotification(notification);
            }, 5000);
        },

        /**
         * Remove notification
         */
        removeNotification(notification: Notification) {
            const index = this.notifications.indexOf(notification);
            if (index > -1) {
                this.notifications.splice(index, 1);
            }
        },

        /**
         * Disconnect WebSocket
         */
        disconnect() {
            if (this.socket) {
                this.socket.disconnect();
                this.socket = null;
                this.connected = false;
            }
        }
    }
});
```

#### 4.2 Update Games Store
Modify `client/src/stores/Games.store.ts`:

```typescript
import { defineStore } from 'pinia';
import { useWebSocketStore } from './WebSocket.store';

export const useGamesStore = defineStore('games', {
    state: () => ({
        games: [] as any[],
        currentGame: null as any,
        // ... existing state
    }),

    actions: {
        /**
         * Update game state from WebSocket
         */
        updateGameState(payload: any) {
            if (this.currentGame?.id === payload.gameId) {
                this.currentGame.turn = payload.turn;
                this.currentGame.world = payload.world;
            }

            // Update in games list
            const index = this.games.findIndex(g => g.id === payload.gameId);
            if (index > -1) {
                this.games[index].turn = payload.turn;
            }
        },

        /**
         * Add player to game
         */
        addPlayerToGame(gameId: number, playerInfo: any) {
            const game = this.games.find(g => g.id === gameId);
            if (game) {
                if (!game.players) game.players = [];
                game.players.push(playerInfo);
            }
        },

        /**
         * Join game (with WebSocket)
         */
        async joinGame(gameId: number, playerId: number) {
            // Call API
            const response = await api.joinGame(gameId, playerId);
            
            // Join WebSocket room
            const wsStore = useWebSocketStore();
            wsStore.joinGame(gameId, playerId);
            
            return response;
        }
    }
});
```
