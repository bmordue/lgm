# Implementation Plan: Replace Actor Placement Algorithm

**Issue ID**: lgm-k92
**Priority**: P2
**Type**: Task
**Dependencies**: None (ready to start)

## 1. Summary

Replace the current actor placement algorithm with a more intelligent system that finds unoccupied 3x3 grid spaces for player actors, preventing overlaps and ensuring fair starting positions.

## 2. Problem

Current actor placement in `api/service/Rules.ts:setupActors()`:

- **Basic Collision Check**: Only checks if area is empty
- **Sequential Search**: May cluster players together
- **No Strategy**: Doesn't consider map balance
- **Fallback Issues**: May fail to place actors in crowded maps

### Current State
```typescript
// Current implementation
for (let row = 0; row <= world.terrain.length - ACTOR_GRID_SIZE && !placed; row++) {
    for (let col = 0; col <= world.terrain[0].length - ACTOR_GRID_SIZE && !placed; col++) {
        if (isAreaEmpty(x, y, ACTOR_GRID_SIZE, existingActorObjects)) {
            placed = true;
        }
    }
}
```

Issues:
- First available spot may not be best
- No consideration for player balance
- No distance between player starting positions

## 3. Proposed Solution

Implement improved placement algorithm:

1. **Shuffled Search**: Randomize position order for fairness
2. **Minimum Distance**: Ensure players start far apart
3. **Terrain Consideration**: Avoid blocked terrain
4. **Fallback Strategy**: Multiple placement strategies

### Phase 1: Create Placement Service (2-3 hours)

Create `api/service/ActorPlacementService.ts`:
```typescript
export interface PlacementResult {
    startX: number;
    startY: number;
    success: boolean;
}

export function findBestPlacement(
    terrain: Terrain[][],
    existingActors: Actor[],
    gridSize: number = 3,
    minDistanceFromOthers: number = 5
): PlacementResult {
    const validPositions: {x: number, y: number, score: number}[] = [];
    
    // Find all valid positions
    for (let row = 0; row <= terrain.length - gridSize; row++) {
        for (let col = 0; col <= terrain[0].length - gridSize; col++) {
            if (isAreaEmpty(row, col, gridSize, existingActors) &&
                isAreaValidTerrain(row, col, gridSize, terrain)) {
                
                const score = calculatePositionScore(row, col, existingActors, minDistanceFromOthers);
                validPositions.push({x: row, y: col, score});
            }
        }
    }
    
    // Sort by score and pick best
    validPositions.sort((a, b) => b.score - a.score);
    
    if (validPositions.length > 0) {
        return {
            startX: validPositions[0].x,
            startY: validPositions[0].y,
            success: true
        };
    }
    
    return { startX: 0, startY: 0, success: false };
}

function calculatePositionScore(
    x: number,
    y: number,
    existingActors: Actor[],
    minDistance: number
): number {
    let score = 100;
    
    // Bonus for distance from other players
    for (const actor of existingActors) {
        const dist = Math.sqrt(Math.pow(x - actor.pos.x, 2) + Math.pow(y - actor.pos.y, 2));
        if (dist < minDistance) {
            score -= (minDistance - dist) * 10;
        } else {
            score += dist;
        }
    }
    
    // Bonus for center positions (more strategic)
    const centerX = Math.floor(terrain.length / 2);
    const centerY = Math.floor(terrain[0].length / 2);
    const distFromCenter = Math.sqrt(Math.pow(x - centerX, 2) + Math.pow(y - centerY, 2));
    score -= distFromCenter * 0.5;
    
    return score;
}
```
