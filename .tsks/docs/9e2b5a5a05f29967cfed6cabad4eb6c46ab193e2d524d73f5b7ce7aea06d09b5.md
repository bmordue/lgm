# Implementation Plan: Expand E2E Test Coverage

**Issue ID**: lgm-e2e
**Priority**: P3
**Type**: Task
**Dependencies**: None

## 1. Summary

Expand end-to-end test coverage to include all major game flows, multiplayer scenarios, combat situations, and edge cases to ensure the complete system works correctly.

## 2. Problem

Current E2E tests (`api/test/e2e.test.ts`) are limited:

- **Basic Flows Only**: Only tests simple game creation/joining
- **No Combat**: Combat scenarios not tested end-to-end
- **No Multiplayer**: Limited multi-player testing
- **No Edge Cases**: Error conditions not tested
- **No Turn Scenarios**: Complex turn interactions untested

### Current State
```typescript

```

## 3. Proposed Solution

Expand E2E coverage to include:

1. **Complete Game Flows**: Full game lifecycle
2. **Combat Scenarios**: Attack, damage, death
3. **Multiplayer**: 2-4 player games
4. **Edge Cases**: Error conditions, validation
5. **Turn Scenarios**: Multiple turns, order submission
