# Implementation Plan: Add Actor Indication to Hex Grid

**Issue ID**: lgm-hex
**Priority**: P3
**Type**: Task
**Dependencies**: None

## 1. Summary

Add visual indicators to the hex grid frontend component to clearly show actor positions, ownership, health status, and selection state.

## 2. Problem

Current hex grid (`client/src/components/HexGrid.vue`):

- **Unclear Positions**: Hard to see which hex has actors
- **No Health Display**: Can't see damage at a glance
- **Selection Ambiguous**: Selected actor not clearly indicated
- **Ownership Unclear**: Hard to distinguish friendly vs enemy

## 3. Proposed Solution

Add visual indicators:

1. **Actor Tokens**: Display actor icons on hexes
2. **Health Bars**: Show current health percentage
3. **Ownership Colors**: Color-code by player
4. **Selection Highlight**: Clear selected actor indicator
5. **Hover Info**: Show details on mouse hover

## 4. Implementation Plan

### Phase 1: Update HexGrid Component (3-4 hours)

```vue
<!-- client/src/components/HexGrid.vue -->
<template>
  <div class="hex-grid">
    <HexCell
      v-for="hex in hexes"
      :key="hex.id"
      :hex="hex"
      :actor="getActorAt(hex)"
      :selected="isSelected(hex)"
      @click="handleHexClick(hex)"
    >
      <template v-if="actor">
        <ActorToken 
          :actor="actor"
          :show-health="true"
          :show-owner="true"
        />
      </template>
    </HexCell>
  </div>
</template>
```

### Phase 2: Create ActorToken Component (2-3 hours)

```vue
<!-- client/src/components/ActorToken.vue -->
<template>
  <div class="actor-token" :class="ownerClass">
    <img :src="actorIcon" class="actor-icon" />
    <div class="health-bar" v-if="showHealth">
      <div class="health-fill" :style="{ width: healthPercent + '%' }" />
    </div>
  </div>
</template>

<script>
export default {
  props: {
    actor: Object,
    showHealth: Boolean,
    showOwner: Boolean
  },
  computed: {
    ownerClass() {
      return `owner-${this.actor.owner}`;
    },
    healthPercent() {
      return (this.actor.health / this.actor.maxHealth) * 100;
    }
  }
};
</script>
```
