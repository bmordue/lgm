# Implementation Plan: Add Combat-Related Unit Actions

**Issue ID**: lgm-vfj
**Priority**: P2
**Type**: Task
**Depends On**: 
- lgm-9u0 (Integrate combat mechanics into game loop)
- lgm-sqo (Implement damage calculation formulas)
- lgm-m0a (Implement weapon range validation logic)

## 1. Summary

Implement combat-related unit actions including attack, take cover, reload, aim, and other tactical maneuvers that units can perform during their turn. This extends the existing movement-only order system to support full combat gameplay.

## 2. Problem

The current order system (`Models.ts:ActorOrders`) only supports basic movement:

```typescript
export enum OrderType {
    MOVE,
    ATTACK  // Defined but minimally implemented
}

export interface ActorOrders {
    actorId: number;
    orderType: OrderType;
    ordersList?: Array<Direction>;  // For MOVE
    targetId?: number;               // For ATTACK
}
```

Current limitations:
- **No Cover Action**: Units can't actively take cover for defense bonus
- **No Reload Mechanic**: Weapons with limited ammo can't be reloaded
- **No Aim Action**: Can't spend turn to improve accuracy
- **No Overwatch**: Can't set units to react to enemy movement
- **No Melee Actions**: No special close combat maneuvers
- **No Surrender/Retreat**: No way to remove units from combat

### Current State
From `api/service/Rules.ts`:
- `applyMovementOrders()` handles MOVE orders
- `applyFiringRules()` handles ATTACK orders (basic implementation)
- No other action types implemented

From `api/config/WeaponsConfig.ts`:
- Weapons have `ammo` property but no reload mechanism
- Ammo is never consumed or tracked

### Impact
Without combat actions:
- Combat is just "move and shoot"
- No tactical depth
- Ammo mechanics don't work
- No defensive gameplay options
- Limited strategic choices

## 3. Proposed Solution

Extend the order system to support comprehensive combat actions:

### New Order Types
```typescript
export enum OrderType {
    MOVE,           // Existing: Move up to 6 hexes
    ATTACK,         // Existing: Attack a target
    RELOAD,         // NEW: Reload weapon
    TAKE_COVER,     // NEW: Gain cover bonus
    AIM,            // NEW: Improve accuracy next turn
    OVERWATCH,      // NEW: Reactive fire
    MELEE_ATTACK,   // NEW: Close combat
    RETREAT,        // NEW: Remove unit from combat
    HOLD            // NEW: Do nothing (end turn)
}
```

### Action System
Each action has:
- **Duration**: How many timesteps it takes
- **Requirements**: What's needed to perform action
- **Effects**: What the action accomplishes
- **Limitations**: When action can't be performed

/**
 * Process an actor's action order
 */
export async function processActorAction(
    actor: Actor,
    order: ActorOrders,
    game: Game,
    world: World,
    allActors: Actor[],
    timestep: number
): Promise<ActionResult> {
    // Check if actor can act
    if (actor.state !== ActorState.ALIVE) {
        return {
            success: false,
            message: 'Actor is dead',
            actionPointsUsed: 0
        };
    }

    if (actor.hasActed) {
        return {
            success: false,
            message: 'Actor has already acted this turn',
            actionPointsUsed: 0
        };
    }

    const actionDef = ACTION_DEFINITIONS[order.orderType];
    
    // Check requirements
    const missingReqs = checkActionRequirements(actor, order, actionDef.requirements);
    if (missingReqs.length > 0) {
        return {
            success: false,
            message: `Missing requirements: ${missingReqs.join(', ')}`,
            actionPointsUsed: 0
        };
    }

    // Process based on action type
    switch (order.orderType) {
        case OrderType.MOVE:
            return await processMoveAction(actor, order, world, allActors);
        
        case OrderType.ATTACK:
            return await processAttackAction(actor, order, world, allActors);
        
        case OrderType.MELEE_ATTACK:
            return await processMeleeAttackAction(actor, order, world, allActors);
        
        case OrderType.RELOAD:
            return processReloadAction(actor, order);
        
        case OrderType.TAKE_COVER:
            return await processTakeCoverAction(actor, order, world, allActors);
        
        case OrderType.AIM:
            return processAimAction(actor, order);
        
        case OrderType.OVERWATCH:
            return processOverwatchAction(actor, order);
        
        case OrderType.RETREAT:
            return await processRetreatAction(actor, order, world, allActors);
        
        case OrderType.HOLD:
            return processHoldAction(actor);
        
        default:
            return {
                success: false,
                message: `Unknown action type: ${order.orderType}`,
                actionPointsUsed: 0
            };
    }
}

                // Would need to check for visible enemies

    }

    return missing;

/**
 * Process ATTACK action
 */
async function processAttackAction(
    actor: Actor,
    order: ActorOrders,
    world: World,
    allActors: Actor[]
): Promise<ActionResult> {
    if (!order.targetId) {
        return {
            success: false,
            message: 'No target specified',
            actionPointsUsed: 0
        };
    }

    const target = allActors.find(a => a.id === order.targetId);
    if (!target) {
        return {
            success: false,
            message: 'Target not found',
            actionPointsUsed: 0
        };
    }

    // Validate attack
    const validation = await validateAttack(actor, target, world);
    if (!validation.valid) {
        return {
            success: false,
            message: `Invalid attack: ${validation.errors.join(', ')}`,
            actionPointsUsed: 0
        };
    }

    // Calculate and apply damage
    const distance = validation.distance;
    const actorTerrain = world.terrain[actor.pos.x]?.[actor.pos.y] || 0;
    const targetTerrain = world.terrain[target.pos.x]?.[target.pos.y] || 0;
    
    const damageCalc = calculateDamage(
        actor,
        target,
        distance,
        actorTerrain,
        targetTerrain,
        validation.hasLineOfSight
    );

    // Apply damage to target
    target.health = Math.max(0, (target.health || 100) - damageCalc.finalDamage);
    
    // Consume ammo
    if (actor.weapon?.ammo) {
        actor.currentAmmo = (actor.currentAmmo || actor.weapon.ammo) - 1;
    }

    // Check for kill
    if (target.health <= 0) {
        target.state = ActorState.DEAD;
    }

    logger.info(util.format(
        'Actor %d attacked actor %d for %d damage (remaining HP: %d)',
        actor.id, target.id, damageCalc.finalDamage, target.health
    ));

    return {
        success: true,
        message: `Attack dealt ${damageCalc.finalDamage} damage`,
        actionPointsUsed: 1,
        stateChanges: { 
            hasActed: true,
            currentAmmo: actor.currentAmmo
        }
    };
}

/**
 * Process MELEE_ATTACK action
 */
async function processMeleeAttackAction(
    actor: Actor,
    order: ActorOrders,
    world: World,
    allActors: Actor[]
): Promise<ActionResult> {
    if (!order.targetId) {
        return {
            success: false,
            message: 'No target specified',
            actionPointsUsed: 0
        };
    }

    const target = allActors.find(a => a.id === order.targetId);
    if (!target) {
        return {
            success: false,
            message: 'Target not found',
            actionPointsUsed: 0
        };
    }

    // Check adjacency (melee range = 1)
    const actorHex = gridPositionToHex(actor.pos);
    const targetHex = gridPositionToHex(target.pos);
    const distance = actorHex.distance(targetHex);

    if (distance > 1) {
        return {
            success: false,
            message: 'Target not adjacent (melee range)',
            actionPointsUsed: 0
        };
    }

    // Melee attacks don't use ammo and have different damage calculation
    // Simplified for now - would use melee-specific damage formula
    const baseDamage = actor.weapon?.damage || 10;
    const damage = Math.max(1, baseDamage);

    target.health = Math.max(0, (target.health || 100) - damage);

    if (target.health <= 0) {
        target.state = ActorState.DEAD;
    }

    return {
        success: true,
        message: `Melee attack dealt ${damage} damage`,
        actionPointsUsed: 1,
        stateChanges: { hasActed: true }
    };
}

/**
 * Process RELOAD action
 */
function processReloadAction(actor: Actor, order: ActorOrders): ActionResult {
    if (!actor.weapon || !actor.weapon.ammo) {
        return {
            success: false,
            message: 'Weapon does not use ammo',
            actionPointsUsed: 0
        };
    }

    const maxAmmo = actor.weapon.ammo;
    const currentAmmo = actor.currentAmmo || 0;
    
    if (currentAmmo >= maxAmmo) {
        return {
            success: false,
            message: 'Weapon already fully loaded',
            actionPointsUsed: 0
        };
    }

    const reloadAmount = order.reloadAmount || maxAmmo;
    const newAmmo = Math.min(maxAmmo, currentAmmo + reloadAmount);

    return {
        success: true,
        message: `Reloaded to ${newAmmo}/${maxAmmo}`,
        actionPointsUsed: 1,
        stateChanges: { 
            hasActed: true,
            currentAmmo: newAmmo
        }
    };
}

// Helper functions
function gridPositionToHex(pos: GridPosition): Hex {
    const q = pos.y;
    const r = pos.x - (pos.y - (pos.y & 1)) / 2;
    return new Hex(q, r, -q - r);
}

function isActorEngaged(actor: Actor, allActors: Actor[]): boolean {
    const actorHex = gridPositionToHex(actor.pos);
    
    return allActors.some(other => {
        if (other.id === actor.id || other.state !== ActorState.ALIVE || other.owner === actor.owner) {
            return false;
        }
        
        const otherHex = gridPositionToHex(other.pos);
        return actorHex.distance(otherHex) <= 1;
    });
}

export async function applyRulesToActorOrders(
    game: Game,
    world: World,
    allActorOrders: Array<ActorOrders>,
    allActorsInWorld: Actor[]
): Promise<Array<Actor>> {
    if (!allActorOrders || allActorOrders.length === 0) {
        logger.warn("applyRulesToActorOrders: did not receive any orders");
        return allActorsInWorld;
    }

    // Reset acted status for new turn
    for (const actor of allActorsInWorld) {
        actor.hasActed = false;
        actor.actionPoints = actor.actionPoints || 1;
    }

    // Process orders by timestep
    for (let ts = 0; ts < TIMESTEP_MAX; ts++) {
        for (const ao of allActorOrders) {
            const actor = allActorsInWorld.find(a => a.id === ao.actorId);
            if (!actor) continue;

            // Process the action
            const result = await processActorAction(
                actor,
                ao,
                game,
                world,
                allActorsInWorld,
                ts
            );

            // Apply state changes
            if (result.success && result.stateChanges) {
                Object.assign(actor, result.stateChanges);
            }

            logger.debug(util.format(
                'Actor %d action %s: %s',
                ao.actorId,
                OrderType[ao.orderType],
                result.message
            ));
        }
    }

    // Reset aim bonus after turn
    for (const actor of allActorsInWorld) {
        if (actor.isAiming) {
            actor.isAiming = false;
            actor.aimBonus = 0;
        }
        
        // Reset overwatch
        if (actor.isOnOverwatch) {
            actor.isOnOverwatch = false;
            actor.overwatchTriggered = false;
        }
    }

    return allActorsInWorld;
}

/**
 * Check and trigger overwatch fire when enemy moves
 */
export async function checkOverwatchTriggers(
    movingActor: Actor,
    fromPos: GridPosition,
    toPos: GridPosition,
    world: World,
    allActors: Actor[]
): Promise<void> {
    // Find all actors on overwatch
    const overwatchActors = allActors.filter(a => 
        a.isOnOverwatch && 
        a.state === ActorState.ALIVE &&
        a.weapon &&
        (a.currentAmmo || 0) > 0
    );

    for (const overwatchActor of overwatchActors) {
        // Check if moving actor is enemy
        if (movingActor.owner === overwatchActor.owner) {
            continue;
        }

        // Check line of sight from overwatch actor to moving actor
        const overwatchHex = gridPositionToHex(overwatchActor.pos);
        const movingHex = gridPositionToHex(movingActor.pos);
        
        const hasLOS = hasLineOfSight(overwatchHex, movingHex, world.terrain, allActors);
        
        if (!hasLOS) {
            continue;
        }

        // Check range
        const rangeCheck = isTargetInRange(
            overwatchActor.pos,
            movingActor.pos,
            overwatchActor.weapon!
        );

        if (!rangeCheck.inRange) {
            continue;
        }

        // Trigger overwatch fire
        await triggerOverwatchFire(overwatchActor, movingActor, world, allActors);
    }
}

/**
 * Execute overwatch attack
 */
async function triggerOverwatchFire(
    overwatchActor: Actor,
    target: Actor,
    world: World,
    allActors: Actor[]
): Promise<void> {
    if (overwatchActor.overwatchTriggered) {
        return;  // Already fired this turn
    }

    const distance = overwatchActor.weapon!.maxRange / 2;  // Simplified
    const actorTerrain = world.terrain[overwatchActor.pos.x]?.[overwatchActor.pos.y] || 0;
    const targetTerrain = world.terrain[target.pos.x]?.[target.pos.y] || 0;

    // Overwatch has accuracy penalty
    const damageCalc = calculateDamage(
        overwatchActor,
        target,
        distance,
        actorTerrain,
        targetTerrain,
        true
    );

    // Apply reduced damage (overwatch penalty)
    const overwatchDamage = Math.floor(damageCalc.finalDamage * 0.7);
    
    target.health = Math.max(0, (target.health || 100) - overwatchDamage);
    
    if (target.health <= 0) {
        target.state = ActorState.DEAD;
    }

    overwatchActor.currentAmmo! -= 1;
    overwatchActor.overwatchTriggered = true;

    logger.info(util.format(
        'Overwatch: Actor %d fired at %d for %d damage',
        overwatchActor.id, target.id, overwatchDamage
    ));
}

## 8. Open Questions

1. **Action Points System**: Should we use a more flexible AP system (e.g., 2-3 AP per turn)?
   - Current: 1 AP per action
   - Alternative: Different actions cost different AP, actors have 2-3 AP

2. **Overwatch Priority**: How to handle multiple overwatch triggers?
   - First-come-first-served?
   - Priority based on actor stats?

3. **Retreat Mechanics**: What happens to retreating units?
   - Removed from game?
   - Move off map edge?
   - Cannot rejoin?

4. **Action Queueing**: Can players queue multiple actions?
   - Current: One action per turn
   - Alternative: Queue MOVE + ATTACK

## 9. Risks and Considerations

### Complexity
Many action types increase code complexity. Mitigation:
- Clear action definitions
- Consistent interface
- Good test coverage

### Balance
Actions need playtesting for balance. Mitigation:
- Configuration-driven values
- Easy to tune
- Document balance assumptions

### Performance
Overwatch checks could be expensive. Mitigation:
- Limit overwatch triggers per turn
- Cache line-of-sight checks
- Profile and optimize
