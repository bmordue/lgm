# Implementation Plan: Implement Database Integration

**Issue ID**: lgm-8dy
**Priority**: P2
**Type**: Feature
**Dependencies**: None (ready to start)

## 1. Summary

Replace the in-memory data storage with PostgreSQL database for persistent storage of games, users, turns, and actors. This is critical for production readiness.

## 2. Problem

Current system uses in-memory storage (`api/service/Store.ts`):

- **No Persistence**: All data lost on server restart
- **Not Scalable**: Memory limits constrain growth
- **No Data Integrity**: Lacks transactional guarantees
- **Cannot Share State**: Multiple server instances can't share data

### Current State
```typescript
// api/service/Store.ts - In-memory only
const store: { [key: string]: any } = {};

export function create(key: string, data: any): number {
    const id = nextId++;
    store[`${key}:${id}`] = data;
    return id;
}
```

## 3. Proposed Solution

Implement PostgreSQL with Prisma ORM:

1. **Database Schema**: Define models for User, Game, Player, Actor, Turn
2. **Prisma Client**: Type-safe database access
3. **Migrations**: Version-controlled schema changes
4. **Service Layer**: Replace Store.ts with database operations

## 4. Implementation Plan

### Phase 1: Setup (2-3 hours)

```bash
cd api
npm install pg

npx prisma init --datasource-provider postgresql
```

Configure `.env`:
```
DATABASE_URL="postgresql://user:password@localhost:5432/lgm?schema=public"
```

### Phase 2: Define Schema (2-3 hours)

Create `prisma/schema.prisma`:
```prisma
generator client {
    provider = "prisma-client-js"
}

datasource db {
    provider = "postgresql"
    url      = env("DATABASE_URL")
}

model User {
    id        Int      @id @default(autoincrement())
    username  String   @unique
    createdAt DateTime @default(now())
    players   Player[]
}

model Game {
    id          Int      @id @default(autoincrement())
    turn        Int      @default(0)
    maxPlayers  Int      @default(4)
    state       GameState @default(LOBBY)
    createdAt   DateTime @default(now())
    startedAt   DateTime?
    players     Player[]
    turns       Turn[]
    world       World?
}

model Player {
    id        Int      @id @default(autoincrement())
    gameId    Int
    userId    Int
    isHost    Boolean  @default(false)
    joinedAt  DateTime @default(now())
    game      Game     @relation(fields: [gameId], references: [id])
    user      User     @relation(fields: [userId], references: [id])
    actors    Actor[]
}

model World {
    id        Int      @id @default(autoincrement())
    gameId    Int      @unique
    terrain   Json
    actors    Actor[]
    game      Game     @relation(fields: [gameId], references: [id])
}

model Actor {
    id       Int     @id @default(autoincrement())
    playerId Int
    worldId  Int
    posX     Int
    posY     Int
    health   Int     @default(100)
    state    String  @default("ALIVE")
    weapon   Json?
    player   Player  @relation(fields: [playerId], references: [id])
    world    World   @relation(fields: [worldId], references: [id])
}

model Turn {
    id      Int      @id @default(autoincrement())
    gameId  Int
    turnNum Int
    orders  Json
    results Json?
    createdAt DateTime @default(now())
    game    Game     @relation(fields: [gameId], references: [id])
}

enum GameState {
    LOBBY
    IN_PROGRESS
    COMPLETED
}
```

Run migration:
```bash
npx prisma migrate dev --name init
```

### Phase 3: Create Database Service (2-3 hours)

Create `api/service/DatabaseService.ts`:
```typescript
import { PrismaClient } from '@prisma/client';

const prisma = new PrismaClient({
    log: ['query', 'info', 'warn', 'error']
});

export async function connectDatabase() {
    await prisma.$connect();
    console.log('Database connected');
}

export async function disconnectDatabase() {
    await prisma.$disconnect();
}

export { prisma };
```

Update `api/index.ts`:
```typescript
import { connectDatabase, disconnectDatabase } from './service/DatabaseService';

connectDatabase();
process.on('SIGINT', async () => {
    await disconnectDatabase();
    process.exit(0);
});
```

### Phase 4: Replace Store.ts (4-6 hours)

Create new service layer to replace Store.ts operations:

```typescript
// api/service/GameRepository.ts
import { prisma } from './DatabaseService';

export async function createGame(data: any) {
    return prisma.game.create({ data });
}

export async function getGame(id: number) {
    return prisma.game.findUnique({ where: { id } });
}

export async function updateGame(id: number, data: any) {
    return prisma.game.update({ where: { id }, data });
}

export async function getGames() {
    return prisma.game.findMany();
}
```
