# Implementation Plan: Enhance Authentication and User Management

**Issue ID**: lgm-2dg
**Priority**: P2
**Type**: Feature
**Dependencies**: 
- lgm-8dy (Implement database integration)

## 1. Summary

Implement a complete authentication system with user registration, password hashing, and JWT-based session management to replace the current placeholder authentication system.

## 2. Problem

Current authentication in `api/controllers/UsersController.ts` is a simple placeholder:

- **No Authentication**: Anyone can impersonate any user with just a username
- **No Registration**: No formal account creation process
- **No Password Protection**: System doesn't handle passwords
- **Insecure Sessions**: Sessions tied to simple in-memory storage

### Current State
```typescript
// Current placeholder - no auth
POST /users/
{
    "username": "player1"
}
// Returns session without any verification
```

### Impact
- No security for user accounts
- No user ownership of games
- Cannot implement user-specific features
- Not production-ready

## 3. Proposed Solution

Implement JWT-based authentication:

1. **User Registration**: Create accounts with username/password
2. **Password Hashing**: Secure storage using bcrypt
3. **JWT Tokens**: Secure session management
4. **Protected Routes**: Middleware for authenticated endpoints

### Phase 3: Create AuthService (2-3 hours)

Create `api/service/AuthService.ts`:
```typescript
import bcrypt from 'bcrypt';
import jwt from 'jsonwebtoken';
import { prisma } from './DatabaseService';

const SALT_ROUNDS = 10;
const JWT_SECRET = process.env.JWT_SECRET || 'dev-secret';
const JWT_EXPIRY = '7d';

export async function register(username: string, password: string): Promise<{ userId: number }> {
    const passwordHash = await bcrypt.hash(password, SALT_ROUNDS);
    
    const user = await prisma.user.create({
        data: { username, passwordHash }
    });
    
    return { userId: user.id };
}

export async function login(username: string, password: string): Promise<{ token: string; userId: number }> {
    const user = await prisma.user.findUnique({ where: { username } });
    
    if (!user) {
        throw new Error('Invalid credentials');
    }
    
    const valid = await bcrypt.compare(password, user.passwordHash);
    if (!valid) {
        throw new Error('Invalid credentials');
    }
    
    const token = jwt.sign(
        { userId: user.id, username: user.username },
        JWT_SECRET,
        { expiresIn: JWT_EXPIRY }
    );
    
    return { token, userId: user.id };
}

export function verifyToken(token: string): { userId: number; username: string } {
    return jwt.verify(token, JWT_SECRET) as any;
}
```

### Phase 4: Create Auth Controller (2-3 hours)

Create `api/controllers/AuthController.ts`:
```typescript
export async function register(context: ExegesisContext) {
    const { username, password } = context.params.body;
    
    try {
        const result = await AuthService.register(username, password);
        return { success: true, userId: result.userId };
    } catch (e) {
        context.res.status(400);
        return { error: e.message };
    }
}

export async function login(context: ExegesisContext) {
    const { username, password } = context.params.body;
    
    try {
        const result = await AuthService.login(username, password);
        return { token: result.token, userId: result.userId };
    } catch (e) {
        context.res.status(401);
        return { error: e.message };
    }
}
```

### Phase 5: Create Auth Middleware (1-2 hours)

Create `api/middleware/auth.ts`:
```typescript
import { verifyToken } from '../service/AuthService';

export function authMiddleware(req: Request, res: Response, next: NextFunction) {
    const authHeader = req.headers.authorization;
    
    if (!authHeader || !authHeader.startsWith('Bearer ')) {
        return res.status(401).json({ error: 'No token provided' });
    }
    
    const token = authHeader.substring(7);
    
    try {
        const payload = verifyToken(token);
        (req as any).user = payload;
        next();
    } catch (e) {
        return res.status(401).json({ error: 'Invalid token' });
    }
}
```
