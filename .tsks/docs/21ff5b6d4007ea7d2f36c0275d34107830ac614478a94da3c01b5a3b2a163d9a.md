# Implementation Plan: Improve Actor Visibility Information

**Issue ID**: lgm-vis
**Priority**: P3
**Type**: Task
**Dependencies**: None

## 1. Summary

Enhance the visibility system to provide better information about which actors are visible to each player, including partial visibility states and redacted enemy information.

## 2. Problem

Current visibility system (`api/service/Visibility.ts`):

- **Binary Visibility**: Actors either fully visible or hidden
- **No Partial Info**: No fog-of-war for partially observed areas
- **Full Enemy Data**: Visible enemies show all stats
- **No Exploration State**: Doesn't track explored vs visible

## 3. Proposed Solution

Improve visibility with:

1. **Visibility States**: Visible, Explored, Unexplored
2. **Redacted Info**: Hide enemy stats unless fully observed
3. **Last Known Position**: Show where enemies were last seen
4. **Detection Range**: Different ranges for different actor types

## 4. Implementation Plan

### Phase 1: Extend Visibility Types (2 hours)

```typescript
export enum VisibilityState {
    VISIBLE = 'visible',      // Currently visible
    EXPLORED = 'explored',    // Seen before, not currently visible
    UNEXPLORED = 'unexplored' // Never seen
}

export interface VisibilityInfo {
    state: VisibilityState;
    lastSeenTurn?: number;
    lastKnownPosition?: GridPosition;
}
```

### Phase 2: Update Visibility Service (3-4 hours)

Enhance `getVisibleWorldForPlayer()`:
```typescript
export function getVisibleWorldForPlayer(
    world: { terrain: Terrain[][], actors: Actor[] },
    playerId: number,
    visibilityHistory: Map<number, VisibilityInfo>
): { terrain: Terrain[][], actorIds: number[], actors: Actor[], visibility: VisibilityInfo[] } {
    // Existing visibility calculation...
    
    // Add explored tiles (seen before but not currently visible)
    const exploredTiles = getExploredTiles(playerId, visibilityHistory);
    
    // Redact enemy actor information
    const actorsWithVisibility = actors.map(actor => ({
        ...actor,
        // Hide stats for enemies
        health: actor.owner === playerId ? actor.health : undefined,
        weapon: actor.owner === playerId ? actor.weapon : undefined
    }));
    
    return { terrain, actorIds, actors: actorsWithVisibility, visibility };
}
```
